# Utils

Helper functions and constants.

## Utilities to Build

### generateId.js
**Prompt:** Create a function `generateId()` that returns a unique ID string (use Date.now() + Math.random()). Export as default.

### constants.js
**Prompt:** Create constants for filter types and localStorage key. Export:
- FILTER_ALL = 'all'
- FILTER_ACTIVE = 'active'
- FILTER_COMPLETED = 'completed'
- STORAGE_KEY = 'todos'

### validation.js
**Prompt:** Create utility functions for input validation. Export `isValidTodoText(text)` that checks if text is not empty and not just whitespace.
