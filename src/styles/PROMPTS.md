# Styles

Global styles and Tailwind configuration.

## Files to Create

### globals.css
**Prompt:** Create a global CSS file imported at the app root. Include:
- Reset margins/padding on html, body
- Set base font family to system stack (sans-serif)
- Base colors: light background (#f5f5f5), dark text (#333)
- Base button and input styles (minimal)

### tailwind.config.js
**Prompt:** Create Tailwind config. Extend colors if needed. Keep it minimal. Point to src directory for content scanning.

### App.css (optional)
**Prompt:** Create component-scoped styles for the main app container if needed. Use Tailwind classes primarily; add custom styles only if necessary.
