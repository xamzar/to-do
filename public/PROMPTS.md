# Public Assets

Static files served directly.

## Files

### index.html
**Prompt:** Create an HTML template with:
- DOCTYPE html
- Meta charset utf-8, viewport for mobile
- Title: "To-Do List"
- Link to Tailwind CSS CDN (or local build)
- Root div id="root"
- Script tag pointing to src/index.jsx (or bundled JS)

### favicon.ico (optional)
Use a simple icon or leave blank.
